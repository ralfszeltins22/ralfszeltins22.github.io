Lightweight

Only ~25 KB of pure CSS—no JavaScript, no trackers, no bloat.

Responsive

Looks great on phones, tablets and widescreen desktops with zero extra work.

Consistent

Unified colors, rounded corners, and smooth hover effects across every page.

How to install in three easy steps

1. Install Stylus for Chrome or Firefox.

2. Click “Download CSS” above and save the file somewhere handy.

3. Open Stylus → “Write style for e-klase.lv” → paste the CSS → save. Done!

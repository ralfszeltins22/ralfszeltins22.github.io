UZMANĪBU: Šis stils ir nepabeigts - sagaidiet kļūdas

E-klase Refresh

Moderns stils Legacy E-klasei (my.e-klase.lv)

Uzstādīšana

1. Instalējiet Stylish paplašinājumu

Vispirms instalējiet pārlūka paplašinājumu:

Firefox paplašinājums Chrome paplašinājums

2. Nokopējiet CSS kodu

Nokopējiet stila kodu zemāk:

eklase_refresh-alpha_0.0.2.css
Ielādē CSS...

3. Izveidojiet jaunu stilu

Stylish paplašinājumā:

• Noklikšķiniet uz paplašinājuma ikonas

• Izvēlieties "Write new style for" vai "Rakstīt jaunu stilu"

• Nosauciet to "E-klases Refresh"

4. Uzstādiet domēnu

• Sadaļā "Applies to" izvēlieties URLs starting with

• Blakus ievadiet: https://my.e-klase.lv

5. Ievadiet kodu

• Ielīmējiet nokopēto CSS kodu

• Noklikšķiniet "Save" vai "Saglabāt"

• Apmeklējiet e-klase.lv un uzklikšķiniet uz "Darbinieks"

Problēmu risināšana

Stils nedarbojas

• Pārbaudiet, vai Stylish paplašinājums ir ieslēgts

• Pārliecinieties, ka "URLs starting with" ir uzstādīts uz "https://my.e-klase.lv"

• Atjaunojiet E-klases lapu

• Pārbaudiet, vai E-Klase Refresh ir ieslēgts

Vizuālās problēmas

• Notīriet pārlūka kešatmiņu

• Atcerieties: šī ir alpha versija - kļūdas ir sagaidāmas

• Mēģiniet atjaunot lapu ar Ctrl+F5 (piespiedu atjaunošana)

Stils daļēji darbojas

• Pārbaudiet, vai visa CSS koda daļa ir pareizi iekopēta

• Mēģiniet izdzēst un pārveidot stilu no sākuma

• Pārliecinieties, ka nav kļūdu CSS kodā

Par stilu

Šī ir alfa versija E-klases stila atjaunojumam. Tas iekļauj pamata uzlabojumus krāsām, fontiem un izkārtojumam. Vairāk funkciju tiks pievienotas turpmākajās versijās.

Pašreizējās funkcijas

• Modernizēti pogu un formu stili

• Uzlabots navigācijas izkārtojums

• Labāka lietotāja saskarnes lasāmība

Plānotie uzlabojumi

• Mobilo ierīču atbalsts

• Tumšā režīma iespēja

• Papildu pieejamības funkcijas

• Kļūdu labojumi un optimizācija
